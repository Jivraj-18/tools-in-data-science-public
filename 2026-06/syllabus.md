# Syllabus (2026)

Below is the full **8-week** course map (dummy pages for now).

## Weeks

- [Week 01 — Development Environment & Tooling](/2026-06/week-1) — Set up a modern dev environment and learn the daily tools used by data/AI engineers: editors, packaging, shell, Git, data formats, and lightweight databases.
- [Week 02 — Deployment & API Engineering](/2026-06/week-2) — Build and ship real APIs with FastAPI, Docker, auth, observability, and caching. Focus is on production-style services and deployment workflows.
- [Week 03 — LLM Engineering](/2026-06/week-3) — Learn prompt + context engineering, structured output, embeddings, search primitives, caching, and evaluation tooling around LLM apps.
- [Week 04 — RAG & Hybrid RAG](/2026-06/week-4) — Design retrieval systems: chunking, vector DBs, hybrid search, reranking, grounding, evaluation, and caching for fast reliable RAG.
- [Week 05 — Agentic AI](/2026-06/week-5) — Build agent systems: planning loops, tool calling, async execution, LangGraph, MCP servers, multi-agent coordination, and evaluation.
- [Week 06 — Web Scraping & Data Processing](/2026-06/week-6) — Scrape, parse, deduplicate, and analyze data: browser automation, crawling, anti-bot strategies, document parsing, DuckDB/Parquet, and multimodal pipelines.
- [Week 07 — CI/CD, Security & Cloud Infrastructure](/2026-06/week-7) — Harden and ship: CI/CD pipelines, container security, LLM security + guardrails, cloud compute basics, IaC, budgets, and event-driven systems.
- [Week 08 — MLOps, Fine-Tuning & Model Publishing](/2026-06/week-8) — Train, fine-tune, package, publish, and monitor models: data/versioning, MLflow, Vertex AI, HuggingFace, quantization, and responsible model release.
