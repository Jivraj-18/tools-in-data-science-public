# Tools in Data Science (TDS) — May 2026

This site is the course portal for **Tools in Data Science** (IIT Madras · BS Data Science Programme · May Term).

- **Duration:** 8 weeks
- **Format:** Lecture + Hands-on Lab (every week)

## What’s new in 2026

This curriculum focuses on the current AI engineering stack: modern developer tooling, API/deployment workflows, LLM systems, RAG, agentic AI, security, and production MLOps.

## How to use this portal

1. Follow the weeks in order (Week 01 → Week 08).
2. Complete the **Hands-on Labs** every week.
3. Publish optional Discourse blogs (non-graded, indexed on the portal).
